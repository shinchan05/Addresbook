
A simple address book project to store and manage contact details.

->> Features
- Add contact
- View contacts
- Search contacts
- Delete contacts
- Edit contacts
  
->> How to Run
1. Clone the repo  
2. Open the project folder  
3. Compile and run the code (based on your language)

->> Folder Structure
- `/src` – source code  
- `/data` – saved contacts  
- `/docs` – documentation (optional)
